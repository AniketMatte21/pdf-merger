# PDF Merger

A simple web-based PDF merger application built with HTML, CSS, and JavaScript.

## Features

- Select multiple PDF files
- Merge them into a single PDF document
- Download the merged PDF
- Responsive design with modern UI
- Progress indicator during merging
- Remove individual files before merging

## How to Use

1. Open `index.html` in your web browser
2. Click "Choose PDF Files" to select multiple PDF files
3. Review the selected files and remove any if needed
4. Click "Merge PDFs" (requires at least 2 files)
5. Wait for the merge process to complete
6. Click "Download Merged PDF" to save the result

## Requirements

- Modern web browser with JavaScript enabled
- No server required - runs entirely in the browser

## Files

- `index.html`: Main HTML structure
- `style.css`: Styling and animations
- `script.js`: PDF merging functionality using pdf-lib

## Note

This application processes PDFs entirely in your browser for privacy. Large files may take time to process.
